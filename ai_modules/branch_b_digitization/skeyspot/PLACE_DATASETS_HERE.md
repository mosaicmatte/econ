# Dataset Location

When setting up this project on a new machine, you must download the massive image datasets from Google Drive and place them in this exact folder.

**Google Drive Link:** [bki_project Datasets](https://drive.google.com/drive/folders/1XePl9ygaadwUn7pL8B_sS2LWTdqR6_z8?usp=sharing)

After downloading, extract them so that the folders look exactly like this:
```
econ/ai_modules/branch_b_digitization/skeyspot/
├── CubiCasa5k/
├── yolo_dataset/
└── PLACE_DATASETS_HERE.md
```
